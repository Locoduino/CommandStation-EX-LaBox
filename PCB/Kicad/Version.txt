Kicad 9.0 is used for the latest versions.
The plugin 'Fabrication Toolkit' dedicated to JLCPCB must be installed. To be able to install it, Kicad must be started in administrator mode.
In the properties (PCB view) of some components which must be assembled by JLCPCB, a particular property 'FT Rotation Offset' has been added to manage a correct orientation of these components (D1, D2, D3, U1, and IC2).

Beware of the manufacture with other compagnies. The production files are build with a JLCPCB dedicated tool. These files could work with other manufacturers, but this is not sure. You must check the correct placement of the components before checkin out the order !
